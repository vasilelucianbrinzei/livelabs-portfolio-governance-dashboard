# Status Legend

## Green

The evidence is aligned. For markdown date checks, the acknowledgement date matches the Git update month.

## Yellow

The item needs review but is not automatically a failure. For markdown date checks, the acknowledgement date is within the current 12-month review threshold, which still needs manager validation.

## Red

The item needs priority attention. For markdown date checks, the acknowledgement date is more than 12 months behind the Git update evidence, or the underlying status indicates a high-risk governance review path.
